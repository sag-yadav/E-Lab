# 🎓 E-Lab Website

An interactive and user-friendly **E-Learning Website** developed using **Advanced Java**, **JSP**, **Servlets**, **HTML**, **CSS**, and **MySQL**.  
This project provides an online education platform where students can access courses, study materials, and learning resources easily. 🚀

---

## ✨ Features

- 👨‍🎓 Student Registration & Login  
- 🛠️ Admin Dashboard  
- 📚 Course Management  
- 🎥 Video & Study Material Upload  
- 💻 Online Learning Interface  
- 🔐 User Authentication & Authorization  
- 📱 Responsive UI Design  
- 🔗 Database Connectivity using JDBC  
- ⏳ Session Management  

---

## 🛠️ Technologies Used

- ☕ Advanced Java  
- 🌐 JSP (Java Server Pages)  
- ⚙️ Servlets  
- 🔌 JDBC  
- 🗄️ MySQL  
- 🎨 HTML5  
- 💅 CSS3  
- ✨ JavaScript  
- 🚀 Apache Tomcat  

---

## 📂 Project Modules

### 👨‍💼 Admin Module
- ➕ Add/Edit/Delete Courses  
- 👥 Manage Students  
- 📤 Upload Learning Materials  
- 📊 Monitor Platform Activities  

### 👨‍🎓 Student Module
- 📝 Register/Login  
- 🔍 Browse Courses  
- 📖 Access Study Materials  
- 🎯 Learn Online

MySQL database is used for storing:

- 👤 User Information  
- 📚 Course Details  
- 📂 Learning Materials  
- 🔑 Login Credentials  

---

## 🧰 Tools & Software

- 🌍 Apache Tomcat Server  
- 🛢️ MySQL Workbench

## 🎯 Purpose of Project

The main purpose of this project is to provide a digital learning platform for students and educational institutions using Java web technologies. 🌟
